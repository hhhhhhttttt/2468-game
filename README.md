## 游戏核心玩法

### 2468规则
- 初始生成2、4、6、8等数字方块
- 合并规则：相同数字相加
  * 2 + 2 = 4
  * 4 + 4 = 8
  * 6 + 6 = 12
  * 8 + 8 = 16
  * 以此类推...
- 目标：达到2468或更高数字

### 操作方式
- **键盘控制**：方向键 ↑↓←→ 或 WASD 键
- **触屏控制**：手指滑动方向
- **暂停**：点击暂停按钮
